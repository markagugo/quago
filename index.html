XUDO

So i was thinking of what i can build with my Android and Data Science skills combined in one project and fortunately i came up with the movie downloading app (Xudo) idea :). And after like 1 month and a week i'm happy i was able to make the project a reality and here i'm gonna explain how this project was built.

In order to proceed with the project, i needed to have some movies data already to build the app so in order to achieve that i had to use web scraping. i scraped NetNaija site to get some movies title, poster, download link, and subtitle link, trailer link the scraped IMDB site to get the other movie details like genre, stars, runtime, description and directors. When that was done i took roughly 2 weeks to organize my data such as filling missing values and other little stuffs using python. And finally when i was done with data cleaning, i then moved onto building the app.

A preview of my dataset when i was done cleaning

Building the app i used a free hosting company(000webhost) to host my movies dataset and used simple php scripts to help me read the movies in different categories from the database and when that was done i used volley library in to retrive the movies in android with java. After that i needed a way to display recomendation for each movies. I tried writting a java program to calculate the similarities of the movies on retriveing them from the database but that was stopping the app because the movies data was much(2,364). So thinking of another way to do it, i finally decided to use python which already has a prebuilt function for calculating the similarity which was more faster and then wrote a simple script to calculate the similarity of the movies based on the main actors, director, genre and then saved it in a json format and uploaded it to firebase RTDB.

Structure of my json file identifying each movie recomendations with a unique name
each movie with 10 different movies under as its recomendation

And when that was done i then used firebase firestore(NoSql) to store users data and movies collection for each user.

firestore collections

That was it building the app i also built an admin backend where i can upload new movies and recomendation to the app using Python #Streamlit
